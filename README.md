## Todo Web Application
-------------------------------------------------------------------------------
Project Requirements:
----
- Get all the users 
- Add a new user
- Update an user
- Delete an user
- Sending mail to a user


#### Yukarıdaki özellikleri uygulayan bir web uygulaması

-------------------------------------------------------------------------------
Tools and technologies used:

- IDE --> Intellij Idea
- MySQL --> 8.0
- Spring Boot 2.5.3
- Spring Framework
- Maven
- java 11
- Spring Data Jpa
- Thymeleaf
- Spring Security
- Spring Mail Sender
